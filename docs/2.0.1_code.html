源码概述

  • 开发环境
IDE为Microsoft Visual C++ 6.0 Professional Edition，在Windows7 Ultimate x64系统下完美编译。
  • 开发语言
基于C++开发框架，但几乎全部以“C风格”写就。
  • 界面库
Platform SDK for Windows User Interface、MFC4.2。
  • 字符集
使用“_UNICODE”或“_MBCS”宏，可编译为宽字符版或多字节版（不论何版本，生成题库文件均完全兼容）。
  • 目录结构
.\Bin
已归档的可执行文件

.\help
html帮助文档

.\lib
依赖的第三方库，包含：
.\lib\libjpeg            jpeg图像库
.\lib\libpng             png图像库
.\lib\original_package   各库的原始压缩包
.\lib\tinyxml            xml读写库
.\lib\zlib               压缩库

.\res
主程序所使用的资源文件

.\src
主程序的全部源码

QuickQuiz.dsp
适用于Visual C++ 6.0的工程文件

QuickQuiz.dsw
适用于Visual C++ 6.0的工作空间文件，包含全部工程
