使用者Q&A

  • 我是否需要此应用程序？

如果您有以下需求，则本应用程序非常适合您：

  1. 不想花钱，不想使用破解版的商业软件。
  2. 应付单位例行考试，作为考生，手中有txt/word格式题库，但感到自测不便，不想麻烦别人给自己提问。
  3. 应付单位例行考试，作为出题人，手中有txt/word格式题库，但感到出题繁琐，花费大量翻找答案的时间。
  4. 应付知识竞赛，作为选手，希望训练自己的抢答能力，不想麻烦别人给自己提问。
  5. 维护题库，但题库中有大量重复题目，剔除麻烦。
  • 如何获取此应用程序？

您可以访问来获取最新的可执行文件。

  • 运行此应用程序需要什么环境？

您需要一台装载有Windows中文版操作系统的计算机来运行此应用程序，无需额外安装其他任何软件，本应用程序双击启动无需安装。

  • 我是否需要付费？

本应用程序永久免费，不会以任何形式推送广告。若发现收费、推送广告、报毒等异常行为，则说明您正在使用的应用程序被恶意篡改，请重新下载，必要时请查毒。

  • 为什么要创建题库？

您对题目的一切管理、学习、自测，均需要将题目“题库化”后才能进行。题库是一个文件，包含了题目信息，创建题库后，您可以将文本形式的题目通过本应用程序添加进题库中。

  • 如何开始创建一个题库？

点击菜单栏“文件——新建题库”，或点击工具栏最左侧图标，可立即创建一个空白题库。

  • 如何将题目添加到题库中？

您可以参照“新建题目”来向题库中加入题目；若您希望快一点，且您手中的题目文本符合特定的格式（此格式很容易满足），可以参照“批量生成”来快速生成题目。

  • 是否可以在选择题选项中添加图片？

暂时没有这个功能，您可以在题干中添加选项图片，然后在选项中写上说明。

  • 是否可以在题目中添加音视频？

暂时没有这个功能。

  • 是否可以在题目中添加数学公式？

暂时没有这个功能，请将公式保存为图片后添加。

  • 如何添加阅读理解、完形填空、案例分析等题目类型？

“题目组”功能可以满足您的需要。

  • 在非只读模式下，为何无法新建题目组？

您只有在左侧窗格中选中题库节点或题目组节点时，“新建题目组”菜单项才有效，否则为禁用状态。

  • 我想把同一类型的题目区分开，例如区分“简答题”和“论述题”，该如何做？

“自定义类型”功能可以满足您的需要。

  • 发现题目中有错字怎么办？

您可以参照“编辑题目”来修改题目。

  • 为何打开题库后无法修改题目？

打开题库后默认为只读模式，请勾选菜单栏“题库——允许编辑”。

  • 带封面的题库无法打开是什么原因？

题库封面是2.x版本的新特性，无法被旧版本应用程序支持；如果带封面的题库被编辑过或在传输过程中被压缩，题库可能发生不可逆的损坏。

  • 是否可以收藏其它题库中的题目？

不可以，收藏的题目必须是题库内的题目。

  • 在右侧窗口中点击标头排序，是否会影响题库内容？

不影响。

  • 如何知道题目的添加顺序？

在主界面右侧窗格点击“序号”标头，即可按照添加顺序对题目进行排序。

  • 如何剔除题库中的重复题目？

您可以参照“选中所有高相似度的重复题目”来选中相似题目，然后删除。

  • 如何将题目导出为文本？

您可以使用“文字复制”功能来实现导出。

  • 忘记了题库的密码怎么办？

没有办法，本题库采用的加密算法强度较高，极难破解，请一定牢记密码。

  • 我不希望在考试时限时，该如何做？

将答题时限设置为0即可。

  • 我希望生成特定题目的试卷，该如何做？

新建一个题库，将您需要生成的题目复制粘贴入新建题库，再生成试卷。

  • 我想打乱选择题的选项顺序，该如何做？

在“考试设置——选择题设置”中完成设置，参见开始考试。

  • 我希望考试按照固定顺序出题，该如何做？

您可以将题目设置为自定义类型，并按顺序设置题号。

  • 生成的HTML格式试卷为何图片无法正常显示？

您可能使用了较旧版本的浏览器，请试用“HTML文件+图片文件形式”选项，参见生成试卷。

  • 如何进行多人的抢答练习？

每人分配一个键盘按键，在抢答时按下自己的按键，窗口中会显示最先按下了哪个键。
